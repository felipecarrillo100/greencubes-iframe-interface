# Sample Host Application

This project demonstrates how to integrate the Iframe Map Application into a host website using the `greencubes-iframe-interface` library. It serves as a reference implementation and development sandbox.

## Local Development

### 1. Start the Sample
From this directory, run:
```bash
npm run dev
```
The host application will open at `http://localhost:5173`.

### 2. Features Demonstrated
- Initial map setup and configuration.
- Projection switching (2D/3D).
- Dynamic layer addition (WMS, Groups).
- Event logging for clicks and selection.
- Layer tree visualization.

## Usage
The sample application is intended for development and testing of the interface library. Refer to `src/main.tsx` for implementation details on how to use the communication API.
